Pada dasarnya, pernyataan if-else üêç adalah salah satu konsep fundamental dalam pemrograman yang berfungsi untuk mengendalikan alur program. Bayangkan seperti sebuah persimpangan di jalan. Anda bisa memilih jalan mana yang akan diambil berdasarkan kondisi tertentu. if-else memungkinkan program Anda "memilih" jalan yang berbeda berdasarkan hasil dari suatu kondisi logis. Bagaimana if-else Bekerja? Setiap kali Anda menggunakan if, Python akan mengevaluasi ekspresi kondisional (seperti umur >= 18). Ekspresi ini akan selalu menghasilkan salah satu dari dua nilai: True (Benar) atau False (Salah). Jika ekspresi menghasilkan True, blok kode di bawah if akan dieksekusi. Jika ekspresi menghasilkan False, Python akan melompati blok kode if dan beralih ke blok else (jika ada). Penting untuk diingat bahwa indentasi (spasi di awal baris) sangat krusial dalam Python. Blok kode yang termasuk dalam if atau else harus memiliki level indentasi yang sama, biasanya 4 spasi.
